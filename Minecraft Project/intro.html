Introduction

Welcome to the introduction section. Here, I will introduce about the Minecraft Project.

Welcome! The Minecraft Project uses the topic "Weather Observation". It's purpose is to let kids to learn weather in a easy and fun way.

Update logs are available at here.

The world was made for learning purposes. The aim is to allow kids to learn weather via Minecraft.

Summary of Virtual Worlds

I have tested Roblox and Minecraft(Education Edition). After doing some research, I realize that Minecraft have a built-in code builder, which allows me to do something just by running a script. So I decided to use Minecraft, because in that way, I can control the weather without doing a lot of things.
